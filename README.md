# 무기 강화 RPG

터미널에서 실행하는 Python 미니 프로젝트입니다.

## 실행 방법

```bash
python main.py
```

## 게임 흐름

```text
무기 강화에서 무기 구입/장착/강화 -> 일반 전투로 골드 획득 -> 보스 레이드 도전
```

처음 실행하면 짧은 게임 설명을 보여주고 닉네임을 입력받습니다.

## 메인 메뉴

```text
1. 프로필 보기
2. 무기 강화
3. 일반 전투
4. 보스 레이드
5. 게임 종료
```

## 주요 기능

- 프로필 보기
- 무기 강화
- 무기 강화 메뉴 안에서 상점과 창고 이용
- 일반 전투
- 보스 레이드
- 보스 레이드 메뉴 안에서 랭킹/기록 보기
- JSON 자동 저장

## 팀 역할

- 최연준: 전체 프로젝트 구조, 메인 메뉴, 저장 기능, 통합
- 최지현: 무기 강화
- 김민기: 일반 전투, 상점/창고 추가 기능
- 이동수: 보스 레이드

## 테스트 실행
