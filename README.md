# 무기 강화 RPG

터미널에서 실행하는 Python 미니 프로젝트입니다.

## 실행 방법

```bash
python main.py
```

## 게임 흐름

```text
상점에서 무기 구입 -> 창고에서 무기 장착 -> 무기 강화 -> 일반 전투로 골드 획득 -> 보스 레이드 도전
```

## 주요 기능

- 프로필 보기
- 무기 강화
- 일반 전투
- 보스 레이드
- 랭킹/기록 보기
- 상점에서 기본 무기 구입
- 창고에서 보유 무기 진열 및 장착
- JSON 저장

## 팀 역할

- 최연준: 전체 프로젝트 구조, 메인 메뉴, 저장 기능, 통합
- 최지현: 무기 강화
- 김민기: 일반 전투, 상점/창고 추가 기능
- 이동수: 보스 레이드

## 테스트 실행

```bash
python -m unittest
```
